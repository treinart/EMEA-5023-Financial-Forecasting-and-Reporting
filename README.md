# EMEA 5023: Financial Forecasting and Reporting

This repository contains coursework for the University of Colorado Boulder course EMEA 5023: Financial Forecasting and Reporting.

The current project is the Module 2 peer-reviewed assignment, Understanding Financial Statements. The report analyzes Trane Technologies plc, listed on the New York Stock Exchange under ticker TT.

## Live Report

View the live report here:

[Trane Technologies Financial Statement Analysis](https://treinart.github.io/EMEA-5023-Financial-Forecasting-and-Reporting/)

The live report is hosted with GitHub Pages from the root index.html file.

## Project Overview

This assignment required selecting a public company, obtaining its most recent Annual Report or Form 10-K SEC filing, reviewing the Income Statement, Balance Sheet, and Statement of Cash Flows, and providing a brief financial analysis.

I selected Trane Technologies because Thermo King is part of Trane Technologies. Thermo King is a primary vendor in my professional work, which makes Trane Technologies more relevant to analyze than a random public company.

The notebook reviews Trane Technologies’ financial position using annual report data, SEC filing data, selected financial ratios, year-over-year ratio trends, and screenshots of the required financial statements.

## Analysis Coverage

The notebook includes:

* Company overview and rationale for selecting Trane Technologies.
* Primary source documents used for the review.
* Screenshots from the 2025 Annual Report and Form 10-K.
* Review of the Income Statement, Balance Sheet, and Statement of Cash Flows.
* 2025 ratio calculations for liquidity, asset management, profitability, debt management, investment, and market value ratios.
* Year-over-year ratio comparison for 2023, 2024, and 2025.
* Final assessment of Trane Technologies’ financial position.

## Analysis Highlights

| Metric                                          |       2025 Result |
| ----------------------------------------------- | ----------------: |
| Net revenues                                    | $21,321.9 million |
| Operating income                                |  $3,967.4 million |
| Net earnings attributable to Trane Technologies |  $2,918.6 million |
| Operating cash flow                             |  $3,194.5 million |
| Current ratio                                   |              1.25 |
| Quick ratio                                     |              0.92 |
| Gross margin                                    |             36.2% |
| Operating margin                                |             18.6% |
| Net margin                                      |             13.7% |
| Diluted EPS                                     |            $12.98 |

The report concludes that Trane Technologies appears financially healthy based on the 2025 financial statements and the three-year trend review. The strongest signals are margin expansion, cash generation, improved liquidity, lower relative leverage, and increasing earnings per share. The main area to continue monitoring is leverage, especially if demand weakens or working capital becomes less efficient.
